👩‍💼 HR Analytics Dashboard |
Built an interactive HR Analytics Dashboard in Power BI to analyze workforce composition, compensation trends, employee demographics, and hiring growth.

💼 What I Did

- Designed workforce KPIs for executive reporting
- Built age segmentation and gender analysis
- Created cumulative hiring trend analysis
- Developed job title distribution insights
- Implemented DAX-based calculations for HR metrics

📊 Key Insights Delivered

- Total Workforce: 161 employees
- Average Salary: $54K
- Majority workforce in 30–35 age group
- 54.66% female workforce
- Consistent hiring growth trend
