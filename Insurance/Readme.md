🚗 Car Insurance Analytics Dashboard – Power BI
📌 Overview

This project is an end-to-end Business Intelligence solution built in Power BI to analyze car insurance performance across financial metrics, demographics, vehicle types, and geographic distribution.

The dashboard demonstrates:

Data modeling

DAX measure creation

Interactive reporting

KPI design

Risk segmentation analysis

🗂️ Data Modeling

Built a star schema data model

Established relationships between:

Claims table

Policyholder demographics

Vehicle information

Date dimension

Implemented calculated columns and measures using DAX
